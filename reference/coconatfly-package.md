# coconatfly: Comparative Connectomics of Public and In Progress Drosophila Datasets

Experimental package to enable comparative connectomics analysis of
multiple Drosophila connectomics datasets, including some private ones
still under development.

## Package Options

- `coconatfly.use_superclass`:

  Logical. If `TRUE`, rename class/subclass/subsubclass columns to
  superclass/class/subclass in
  [`cf_meta`](https://natverse.org/coconatfly/reference/cf_meta.md),
  [`cf_partners`](https://natverse.org/coconatfly/reference/cf_partners.md),
  and
  [`cf_partner_summary`](https://natverse.org/coconatfly/reference/cf_partner_summary.md).
  Default: `FALSE`. Set with
  `options(coconatfly.use_superclass = TRUE)`.

- `coconatfly.harmonise_class`:

  Logical. If `TRUE`, harmonise class values to malecns style across all
  datasets. Default: `FALSE`. Set with
  `options(coconatfly.harmonise_class = TRUE)`.
